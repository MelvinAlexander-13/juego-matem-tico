## 🚀 Descripción

**Tres en Raya PWA** es una aplicación web progresiva diseñada para ofrecer una experiencia moderna, rápida e intuitiva del tradicional juego de estrategia. El proyecto combina tecnologías web actuales para proporcionar una interfaz atractiva, adaptable y accesible desde cualquier dispositivo.

Gracias a la implementación de características PWA, los usuarios pueden instalar la aplicación directamente desde el navegador y disfrutar de una experiencia similar a la de una aplicación móvil nativa.

---

## 🎯 Propósito del Proyecto

Este proyecto tiene como objetivo demostrar la implementación práctica de tecnologías web modernas mediante el desarrollo de una aplicación interactiva y multiplataforma. Además de ofrecer entretenimiento, busca servir como ejemplo de buenas prácticas en el desarrollo de Aplicaciones Web Progresivas.

## ✨ Efectos Positivos

### 🧠 Desarrollo del pensamiento estratégico

Puede ejecutarse en computadoras, tabletas y dispositivos móviles sin requerir instalaciones complejas.
